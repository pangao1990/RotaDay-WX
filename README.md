# 班妥了微信小程序版

这是基于鸿蒙版“班妥了”重新实现的原生微信小程序，无需 npm 构建即可在微信开发者工具中导入。

## 已实现

- 个人版：六行月历、四种循环模板、单日覆盖、恢复循环、跨午夜工时、下次休息。
- 团队版：成员与岗位、逐日排班、复制/粘贴当天班表、安全清空、成员负荷统计。
- 统计：个人与团队的月度/年度摘要、班次分布、每日密度、年度节奏、团队 CSV 导出。
- 班次：预设班次改名、简称、颜色、起止时间和休息时长；自定义班次新增与安全删除。
- 外观与内容：晴和、夜航、青柚、暖阳、桃晴、轻虹、松风、自定义共八套主题；“今日片刻”完整保留鸿蒙版 7 个个人场景和 7 个团队场景，共 1680 种标题与正文组合，近期 30 条去重，并复用 8 张轻量插画控制包体。
- 数据：全量 JSON 备份导出、从剪贴板覆盖恢复、本机数据重置。
- 微信能力：系统原生导航栏与右上角胶囊、官方自定义 tabBar、微信分享菜单、原生客服会话、原生意见反馈，以及剪贴板读取和相册写入的按需授权。

## 运行

1. 打开微信开发者工具，选择“导入项目”。
2. 项目目录选择本目录。
3. 工程已配置 AppID `wx66f9037b33c344da`，确认当前开发者账号拥有该小程序权限。
4. 点击编译即可运行；提审前按 [`RELEASE_CHECKLIST.md`](./RELEASE_CHECKLIST.md) 完成后台配置和真机验收。

可在终端执行：

```bash
npm test
npm run check
```

## 目录

```text
pages/index/         个人与团队排班首页、编辑弹层、今日片刻
pages/statistics/    月度与年度统计、团队 CSV
pages/settings/      模式、主题、日历、班次、循环与备份
utils/               日期、排班算法、主题、本地存储和内容引擎
custom-tab-bar/      微信官方自定义 tabBar（主题跟随、系统标签页切换）
components/          隐私授权等通用组件
assets/              Logo 与复用的片刻插画
tests/               核心算法和项目静态校验
```

## 数据与隐私

排班、成员和设置默认只保存在当前微信设备的本地缓存中。小程序不依赖业务服务器，也不会在后台持续运行。只有用户主动恢复备份时才读取剪贴板，主动保存片刻海报时才请求写入相册；拒绝授权不会影响其他排班功能。鸿蒙版的系统通知、桌面卡片、文件选择器和应用市场跳转未照搬到小程序，相关场景均改为微信支持且可审核的原生能力。
